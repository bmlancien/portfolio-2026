⚡ SUPER QUICK SETUP
────────────────────

If you just want to install and run:

  1. npm install
  2. Add public/images/logo.png
  3. Uncomment <img> in Navbar.jsx line 16
  4. Uncomment <img> in Footer.jsx line 19
  5. Create public/images/projects/ folder
  6. Add 4 project images
  7. Uncomment <img> in ProjectCard.jsx line 17
  8. npm run dev

Done! 🎉
